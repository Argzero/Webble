HOW TO PLAY WEBBLE:
{

1. Go to https://processing.org/download/ and get the latest version of Processing

2. Unzip the download if it's zipped.

3. Double click on the processing.exe file once you've finished extracting the zip folder

4. Under "File", select "Open".

5. Open the file, "SpiderWeb.pde" from the SpiderWeb folder included here (make sure not to rename the folder or move any of the files in the folder to any other location)

6. Once Webble has loaded into processing.exe, click the Play button. (the arrow in the upper left)

7. Enjoy!

}
